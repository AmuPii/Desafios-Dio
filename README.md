#Desafio Dio: Dashboard de Vendas no Excel

## Descrição do Projeto
Este projeto consiste na criação de um dashboard interativo de vendas no Microsoft Excel. O objetivo é transformar dados brutos de vendas em visualizações claras, permitindo análise de desempenho, tendências e tomada de decisões baseadas em dados.

### Principais Funcionalidades
- KPIs principais: Total de Vendas, Lucro Total, Vendas Médias.
- Gráficos: Vendas por Região, por Categoria, Tendência Mensal, Top Produtos.
- Filtros interativos: Slicers por Região, Categoria, Segmento e Modo de Envio.
- Timeline para filtrar por período.

## Dados Utilizados
- Dataset fictício de vendas de uma superloja (baseado em exemplos públicos como "Sample Superstore Sales").
- Colunas principais: Order ID, Order Date, Region, Category, Sub-Category, Sales, Quantity, Discount, Profit.
- Fonte de inspiração: Datasets públicos disponíveis no Kaggle ou Tableau Community (ex: Sample - Superstore Sales).

O dataset está incluído na aba "Dados" do arquivo Excel.

## Instruções para Reprodução
1. Baixe o arquivo `Dashboard_Vendas.xlsx`.
2. Abra no Microsoft Excel (versão 2016 ou superior recomendada para slicers e timelines).
3. Interaja com os slicers e timeline para filtrar os dados.
4. Para editar: Vá para as abas "Dados" e "Pivot" para modificar as tabelas dinâmicas.

## Tecnologias Utilizadas
- Microsoft Excel (Tabelas Dinâmicas, Gráficos, Slicers, Timeline).
